## Pages
- **Upload (`/`)**: upload dataset, see basic summary and CTA to analyze.
- **Results (`/results`)**: list of alerts, insights, recommendations, key charts.
- **History (`/history`)**: table of past reports with quick summary.

## Core Components
- **FileUploader**: drag & drop / file input for CSV / XLS.
- **AlertCard**: short, critical findings.
- **InsightCard**: metric-driven stories about what changed.
- **RecommendationCard**: what to do next and why.
- **LoadingState**: spinner + message during analysis.
