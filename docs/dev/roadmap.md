# Melytix — MVP Roadmap

## Phase 1 — Skeleton (current)
- Next.js app router structure (`app`, `components`, `services`, `utils`)
- Upload page with basic file uploader
- Results / History pages stubs
- API endpoints `/api/analyze` and `/api/save` (stubs)

## Phase 2 — Parsing & Basic Insights
- Robust CSV parsing
- Basic metric comparison engine
- Static, rule-based alerts & insights

## Phase 3 — AI Layer
- LLM prompts for insight generation
- AI-generated recommendations
- Prompt presets saved in `docs/dev/prompts.md`

## Phase 4 — Persistence & Sharing
- Storage for history (local + remote)
- Simple shareable links or exports
- Authentication (if needed)
